# Tailwind-starter
Instantly start working on your next Tailwindcss project
## Input Path
`assets/src/css/tailwind.css`
## Output Path
`assets/dist/css/main.css`
## Build
`npx tailwindcss -i ./assets/src/css/tailwind.css -o ./assets/dist/css/main.css --watch`
